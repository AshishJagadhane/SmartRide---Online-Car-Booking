# 🚗 SmartRide - Online Car Booking System

SmartRide is an online car booking platform that enables users to browse available vehicles, make reservations, and manage bookings through a simple and user-friendly interface. The system is designed to streamline the car rental process while providing administrators with tools to efficiently manage vehicles, users, and bookings.

## ✨ Features

### User Features
- User Registration and Login
- Browse Available Cars
- View Vehicle Details
- Online Car Booking
- Booking Management
- Booking History
- Responsive User Interface

### Admin Features
- Admin Dashboard
- Manage Vehicles
- Add, Update, and Delete Cars
- Manage User Accounts
- View and Manage Bookings
- Monitor System Activity

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/AshishJagadhane/SmartRide---Online-Car-Booking.git
```

2. Navigate to the project directory

```bash
cd SmartRide---Online-Car-Booking
```

3. Move the project folder to your web server directory (htdocs for XAMPP).

4. Create a MySQL database.

5. Import the database SQL file into MySQL.

6. Start Apache and MySQL services.

7. Open your browser and visit:

```text
http://localhost/SmartRide---Online-Car-Booking
```
